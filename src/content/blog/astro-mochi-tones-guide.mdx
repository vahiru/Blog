欢迎使用 **Astro Mochi Tones** 主题！这是一个基于 Astro 构建的博客主题，完美复刻了 [Stellaris](https://github.com/chiyuki0325/hexo-theme-stellaris) 主题的优雅设计与强大组件。

项目地址：[GitHub - vahiru/Astro-Mochi-Tones](https://github.com/vahiru/Astro-Mochi-Tones)

> **⚠️ 警告**
> 
> 本组件库目前处于 **早期开发阶段 (Alpha)**，可能会包含不可预料的错误。
> 组件库包含大量未经审计的 **AI 生成代码**，请谨慎使用。

## 🚀 快速开始

### 安装

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建

```bash
npm run build
```

### 文本装饰

```astro

<Psw>悬停显示隐藏内容</Psw>
<Kbd>Ctrl</Kbd> + <Kbd>S</Kbd>
<Emp>重点词汇</Emp>
<U>下划线文本</U>

<Del>已废弃内容</Del>
```

- **按键**: <Kbd>Ctrl</Kbd> + <Kbd>S</Kbd>
- **着重号**: <Emp>重点词汇</Emp>

- **波浪线**: <Wavy>拼写错误标识</Wavy>
