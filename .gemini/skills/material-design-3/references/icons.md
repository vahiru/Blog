# M3 图标与 Material Symbols

M3 推广使用 **Material Symbols**，这是一套支持可变轴调节的图标系统。

## 可变轴 (Variable Axes)
1. **Fill (填充):** 0 (线性) 或 1 (面性)。用于表达状态切换（如收藏/取消收藏）。
2. **Weight (字重):** 100 到 700。标准通常为 400。
3. **Grade (级度):** -25 到 200。在不改变图标外框大小的情况下，微调图标线条的粗细，常用于反白显示或高亮。
4. **Optical Size (视觉尺寸):** 20px 到 48px。确保小尺寸下图标清晰，大尺寸下细节精致。

## 布局规格
- **网格:** 24x24dp。
- **安全区:** 20x20dp。
- **线框:** 图标线条应与 2dp 或 1.5dp 的字重保持视觉平衡。
